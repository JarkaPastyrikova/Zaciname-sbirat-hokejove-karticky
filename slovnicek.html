Slovníček

Je to vlastně stejné, jako když člověk začíná s jakýmkoliv jiným koníčkem. Na začátku naráží na nové pojmy, ve kterých se musí zorientovat.

Tady jsou ty základní:

Balíček

Malý set kartiček – obvykle 5 až 12.

FAT Balíček

Větší balíček – často s lepší šancí na vzácnější karty.

Box

Krabička s více balíčky. Běžně obsahuje 20+ balíčků.

Blaster box

Menší box, určený pro hobby sběratele. Cenově dostupnější.

Case

Celá krabice boxů. Profesionální sběratelé ji kupují často.

Base set / Základní set

Základní sada karet z dané edice.

BV

Book value – katalogová hodnota karty.

Wantlist

Seznam karet, které sběratel ještě hledá.

Checklist

Seznam všech karet z dané edice, podle kterého sběratel kontroluje svou sbírku.
